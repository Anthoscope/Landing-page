🍒 Anthoscope Landing Page

🚀 Quick Start
Prerequisites
Node.js (v18 or higher)

npm (v9 or higher)

Installation Steps

1. Clone the repository:
```bash
git clone https://github.com/Anthoscope/Landing-page
cd Landing_page
```

2.Install dependencies:
```bash
npm install
```

3.Start the development server:
```bash
npm run dev
```

4.Open your browser: http://localhost:5173



-----------
Built with: React 19 + Vite + Tailwind CSS
